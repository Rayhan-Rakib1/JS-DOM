What is the Document Object Model(DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document (usually HTML or XML) as a tree of objects, allowing scripts to manipulate the content, structure, and style of a webpage.

How do you select an element from the DOM ?

You can select elements from the DOM using various methods provided by JavaScript.Selects an element by its unique ID.Selects all elements with a specified class name.Selects all elements with a specified tag name.elects the first element that matches a specified CSS selector.

What is event delegation in the context of the DOM ?

Event delegation is a technique in JavaScript where a single event listener is added to a parent element instead of adding individual listeners to each child element. This approach leverages the event bubbling mechanism in the DOM, allowing events to be captured as they propagate up the hierarchy from the target element to the parent

How do you manipulate an element's attributes and styles using the DOM ?

You can manipulate an element's attributes and styles using various methods and properties provided by the DOM in JavaScript.Getting an Attribute Use the getAttribute method to retrieve the value of a specific attribute.Setting an Attribute Use the setAttribute method to change the value of an attribute.Removing an Attribute Use the removeAttribute method to remove an attribute from an element.Using Properties For certain standard attributes (like id, class, src, etc.), you can directly manipulate properties.
